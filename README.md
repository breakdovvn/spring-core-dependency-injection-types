## Описание
Учебный проект на Spring Core, демонстрирующий автоматическое связывание бинов с помощью Dependency Injection, работу аннотаций @Autowired и @Qualifier, а также поведение контейнера при конфликте бинов одного типа.
## Используемые технологии
- Java
- Spring Core
- Dependency Injection (setter / field / constructor / @Bean method)
- Maven
- JUnit
